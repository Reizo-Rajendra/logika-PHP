# PHP Conditional Logic – Jujutsu Kaisen Theme

Project ini merupakan latihan dasar pemrograman **PHP** yang berfokus pada penggunaan berbagai jenis **pengkondisian** dalam program.

Contoh kasus dalam program ini menggunakan tema dari dunia **Jujutsu Kaisen** untuk menggambarkan bagaimana suatu kondisi dapat menentukan aksi yang berbeda dalam kode program.

---

## Fitur Pengkondisian

Program ini menampilkan beberapa jenis pengkondisian yang umum digunakan dalam PHP:

### 1. Kondisi Positif (if)
Program mengecek apakah kutukan muncul.  
Jika kutukan muncul, maka **Yuji Itadori** akan mengusir kutukan tersebut.

### 2. Kondisi Negatif
Program mengecek apakah energi kutukan tidak mencukupi.  
Jika energi tidak cukup, maka **Megumi Fushiguro** tidak dapat memanggil shikigami.

### 3. Dua Kondisi (if – else)
Program mengecek kekuatan kutukan.  
Jika kutukan sangat kuat maka **Satoru Gojo** akan turun tangan, jika tidak maka **Yuji Itadori** akan melawannya sendiri.

### 4. Lebih dari Dua Kondisi (if – elseif – else)
Program menentukan tindakan berdasarkan **tingkat bahaya kutukan**, mulai dari:
- Tingkat rendah
- Tingkat menengah
- Tingkat tinggi
- Tingkat khusus
- Kutukan dari jari Ryomen Sukuna

### 5. Pengkondisian Bersarang (Nested If)
Program mengecek **jenis jurus** dan **jumlah energi** untuk menentukan hasil dari jurus yang digunakan.

### 6. Pengkondisian dengan AND (&&)
Program mengecek dua kondisi sekaligus:
- kekuatan musuh
- situasi pertempuran

### 7. Pengkondisian dengan OR (||)
Program mengecek apakah ancaman berasal dari:
- kutukan tingkat khusus  
- jari Ryomen Sukuna

Jika salah satu kondisi terpenuhi, maka semua penyihir harus bersiap.
